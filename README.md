# IP Blinker 🚀

**Automated Tor IP Rotator** - Changes IP every **45 seconds** forever!

- ✅ **Auto-installs** Tor + curl
- 🔄 **80 IP changes per hour**
- 👀 **Live OLD vs NEW IP display** 
- 🛠️ **Self-fixing** (handles errors)
- 📱 **Student-friendly** demo

## 🚀 Quick Start (30 seconds)
```bash
chmod +x ip_blinker.sh
./ip_blinker.sh
```
